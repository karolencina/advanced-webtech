To run the examples you need to be in a folder with an active virtualenv containing a Flask installation, then:

```
$ export FLASK_ENV=development
$ export FLASK_APP=REPLACE-WITH-NAME-OF-PYTHON-FILE.py

$ python3 -m flask run
```
